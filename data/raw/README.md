# Raw data

## Starter file (Phase 1)

- **`sample_accidents.csv`** — Small synthetic CSV so you can run Phases 2–5 locally without downloading anything.

## Replacing with a real public dataset

1. Download **one** open dataset (e.g., government road safety CSV or a Kaggle “traffic accidents” set).
2. Save it here as `accidents.csv` (or update your scripts to point at the new filename).
3. Note the **license** and **source URL** in the project root `README.md`.

Large files: add the filename to `.gitignore` if you do not want it in git.
